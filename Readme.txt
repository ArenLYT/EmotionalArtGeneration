The training code for the Diffusion and transformer models 
are available in Diffusion.ipynb and Transformer.ipynb respectively.

The "Run Models.ipynb" file contains the code to classify on specific 
images and the code to generate images from the diffusion model.
(Please note this takes some time 
~2 hours to generate 100 images for each class)

install the necessary packages using:

pip install fastai==2.7.9
pip install diffusers
pip install transformers

Dataset request from https://www.artemisdataset-v2.org/
